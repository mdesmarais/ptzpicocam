classmethod create_pan_tilt_packet(cls, buffer, pan_speed, pan_direction, tilt_speed, tilt_direction)¶

Creates a packet for moving the camera around two axes (pan / title).

The given buffer must have a capacity of at least 7 bytes. It will raise a RuntimeError if the buffer is too small. It could have been modified even if an exception is thrown.

Pan speed must be a value between 1 and 24. Tilt speed must be a value between 1 and 23. If any of the previous conditions is not met then a ValueError exception will be raised.

Parameters

  • buffer (bytearray) – a buffer for holding the packet

  • pan_direction (PanDirection) – a direction around the X axe

  • pan_speed (int) – a speed value between 0 and 24

  • tilt_speed (int) – a speed value between 0 and 23

  • tilt_direction (TiltDirection) – a direction around the Y axe

Raises

  • RuntimeError – if the buffer is too small

  • ValueError – if a speed is not valid or the buffer is too small

Returns

a packet containing a drive command

Return type

ptzpicocam.visca.RawViscaPacket

classmethod create_memory_packet(cls, buffer, position_index, memory_action)¶

Creates a packet for interacting with the camera positions memory.

The given buffer must have a capacity of at least 5 bytes. It will raise a RuntimeError if the buffer is too small. It could have been modified even if an exception is thrown.

The camera supports 6 positions : a position must be called by its index. It is an integer between 0 and 5 included.

Parameters

  • buffer (bytearray) – a buffer for holding the packet

  • position_index (int) – index of the position to select (0-5)

  • memory_action (MemoryAction) – action to perform on the selected position

Raises

  • RuntimeError – if the buffer is too small

  • ValueError – if the index is not valid

Returns

a memory packet

Return type

ptzpicocam.visca.RawViscaPacket
