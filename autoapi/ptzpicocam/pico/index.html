class ptzpicocam.pico.Joystick(min_adc_val, max_adc_val, deadzone)¶

Structure containing informations about a 3-axes joystick.

Creates a new joystick structure.

Parameters

  • min_adc_val (int) – minimal value returned by the device ADC

  • max_adc_val (int) – maximal value returned by the device ADC

  • deadzone (int) – deadzone in percentage based on the max_adc_val param

compute_bounds(self)¶

Computes left and right bounds according to the deadzone.

Attributes left_limit and right_limit will be set.

Return type

None
